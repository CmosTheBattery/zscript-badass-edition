## building on linux

### prerequisites

- deutex
- a computer (with linux)
- functioning brain
- keyboard

### guide

1. clone the repository

```
git clone https://github.com/freedoom-badass-edition/freedoom-badass-edition
```

2. run the script

```
freedoom-badass-edition/build.sh
```
